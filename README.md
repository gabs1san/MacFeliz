# 🍔 MacFeliz
*AINDA EM DESENVOLVIMENTO* 

Sistema web desenvolvido para gerenciamento de hamburgueria utilizando ASP.NET MVC.

O projeto tem como objetivo realizar o controle de produtos, clientes e pedidos através de uma aplicação organizada e responsiva, aplicando conceitos de desenvolvimento backend e frontend com .NET.
---
# 🚀 Tecnologias Utilizadas

- C#
- ASP.NET MVC
- Entity Framework
- SQL Server
- HTML5
- CSS3
- Bootstrap
---
# ⚙️ Funcionalidades
- Cadastro de produtos
- Cadastro de clientes
- Gerenciamento de pedidos
- Integração com banco de dados
- Interface responsiva
